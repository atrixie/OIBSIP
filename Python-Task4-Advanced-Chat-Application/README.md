# 💬 Advanced Chat Application

A real-time desktop chat application built using **Python**, **Tkinter**, and **Socket Programming**. This application follows a client-server architecture that allows multiple users to communicate instantly through a clean graphical interface.

---

## 🚀 Features

- 💬 Real-time messaging
- 👤 Username-based login
- 🖥️ Modern Tkinter GUI
- 🧵 Multi-threaded client-server communication
- 📜 Chat history saving and loading
- 😊 Emoji support
- 🔔 Notification sound for new messages
- ⌨️ Press Enter to send messages
- 🧹 Clear chat functionality
- ⚡ Fast local communication using sockets

---

## 🛠️ Technologies Used

- Python
- Tkinter
- Socket Programming
- Threading

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/atrixie/OIBSIP.git
```

### Navigate to Project

```bash
cd Python-Task4-Advanced-Chat-Application
```

### Create Virtual Environment

```bash
python -m venv .venv
```

### Activate Virtual Environment

Windows

```bash
.venv\Scripts\activate
```

Linux / macOS

```bash
source .venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Application

### Start the Server

```bash
python server.py
```

### Start Client 1

```bash
python client.py
```

### Start Client 2

```bash
python client.py
```

Enter different usernames and start chatting.
