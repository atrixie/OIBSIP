# 🩺 Advanced BMI Health Tracker

An advanced desktop BMI (Body Mass Index) calculator developed using **Python**, **Tkinter**, **SQLite**, and **Matplotlib**. This application allows users to calculate their BMI, save records, view history, analyze statistics, visualize BMI trends, and export data to CSV.

---

## 🚀 Features

- Modern GUI using Tkinter
- BMI Calculation
- BMI Category Detection
- Personalized Health Tips
- Input Validation
- SQLite Database Integration
- View BMI History
- Statistics Dashboard
- BMI Trend Graph
- Export History to CSV
- Automatic Date & Time Logging

---

## 📊 BMI Categories

| BMI | Category |
|------|----------|
| Below 18.5 | Underweight |
| 18.5 - 24.9 | Normal Weight |
| 25 - 29.9 | Overweight |
| 30 and above | Obese |
