# 🤖 MAX AI - Intelligent Voice Assistant

An intelligent desktop voice assistant built with Python that performs everyday tasks using voice commands. MAX combines speech recognition, text-to-speech, Gemini AI, weather updates, email automation, reminders, desktop application launching, and customizable commands to create a smart personal assistant.

---

## ✨ Features

- 🎤 Voice Command Recognition
- 🗣️ Text-to-Speech Responses
- 🤖 Gemini AI Integration
- 🌦️ Live Weather Updates
- 📧 Send Emails using Voice
- ⏰ Voice-Based Reminders
- 🔍 Google Search
- 🌐 Open Websites
- 💻 Launch Desktop Applications
- ⚙️ Custom Commands using config.json
- 🕒 Tell Current Time & Date
- 📁 Modular Python Architecture
- 🎯 User-Friendly Console Interface

## Installation

Clone the repository

```bash
git clone https://github.com/atrixie/Python-Task1-MAX-AI-Voice-Assistant.git

cd Python-Task1-MAX-AI-Voice-Assistant
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env`

```env
EMAIL=
PASSWORD=
GEMINI_API_KEY=
WEATHER_API_KEY=
```

Run

```bash
python main.py
```
